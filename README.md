# JaySoftShine - Business Consultant Portal

A modern, full-stack web portal for tax and business consulting services.

## 🎯 Features

### User Portal
- ✅ Browse services (ITR, GST, Compliance, etc.)
- ✅ Place orders with multi-file uploads
- ✅ Secure JWT authentication
- ✅ Order tracking & history
- ✅ Razorpay payment integration
- ✅ PDF receipt generation

### Admin Dashboard
- 📊 Analytics & reporting
- 🛍️ Service management
- 📦 Order management
- 👥 User management
- 💰 Payment tracking
- 📧 Email notifications

## 🛠️ Tech Stack

**Backend**: Node.js, Express, MongoDB
**Frontend**: Next.js, React, Tailwind CSS
**Payments**: Razorpay
**Email**: Zoho SMTP

## 📦 Installation

### Backend
```bash
cd backend
npm install
cp .env.example .env
npm run dev
```

### Frontend
```bash
cd frontend
npm install
cp .env.example .env.local
npm run dev
```
